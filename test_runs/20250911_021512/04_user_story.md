### Title:

User Story: Displaying Chat History Grouped by Date with Per-Item Downloads

### Description:

As a user, I want to view my chat history grouped by date with the option to download individual items, so that I can easily review and manage my conversations.

### Acceptance Criteria:

- The application displays the chat history in chronological order, grouped by date.
- Each group of chats is displayed with a clear header indicating the date range.
- Users can click on an individual chat item to view its contents.
- Users can download each individual chat item as a file (e.g., text or PDF).
- The download feature works seamlessly and does not compromise the security or integrity of the chat content.
