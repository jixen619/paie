### Title:

Exporting BI Datasets to CSV

### Description:

As a user, I want to be able to export my Business Intelligence (BI) dataset interactions, models, and latency data to a CSV file, so that I can easily analyze and visualize the data in a spreadsheet or other tool.

### Acceptance Criteria:

- The exported CSV file contains all required fields from the BI dataset.
- The CSV file is well-formatted with clear column headers.
- The export process does not alter the original dataset.
- The user can choose to export only specific datasets or all available datasets.
