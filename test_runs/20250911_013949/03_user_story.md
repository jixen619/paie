### Title:

Exporting BI Datasets to CSV

### Description:

As a user, I want to be able to export my Business Intelligence (BI) dataset interactions, models, and latency data to a CSV file, so that I can easily analyze and share the data with others.

### Acceptance Criteria:

- The exported CSV file contains all relevant interaction, model, and latency data.
- The CSV file is well-formatted and easy to read.
- The export process does not compromise the integrity of the original dataset.
- The user is able to specify which datasets they want to export.
